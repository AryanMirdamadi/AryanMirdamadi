## 🦖 درباره من

من **آرین میردامادی** هستم، یه توسعه‌دهنده وب و عاشق داده‌ها!  
همیشه دنبال این هستم که با **کد** و **داده** مشکلات رو حل کنم و فرآیندها رو اتوماتیک کنم.

- 🧠 **طراحی و توسعه وب** (Frontend + Backend)
- 🤖 **اتوماسیون فرآیندها** با Python
- 📊 **داشبوردهای مدیریتی** پیشرفته و تعاملی
- 🎯 عاشق **UI/UX**، **Performance** و **Scalability**
